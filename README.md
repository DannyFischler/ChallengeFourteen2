# Tech Blog CMS

## Introduction
Tech Blog CMS is a content management system designed specifically for tech enthusiasts who love writing about technology, programming, and digital trends. This platform enables you to easily publish articles, blog posts, and share your thoughts and opinions with a wide audience.

## Features
- **User Authentication:** Secure sign-up and login functionality.
- **Homepage:** Displays all existing blog posts.
- **User Dashboard:** Personalized space to create, update, and delete your blog posts.
- **Interactive Posts:** Allows users to comment on blog posts.
- **Responsive Design:** Optimized for all devices.

## Technologies Used
- Frontend: HTML, CSS, JavaScript
- Backend: Node.js, Express.js
- Database: MongoDB
- Authentication: Passport.js


### Prerequisites
- Node.js
- NPM (Node Package Manager)
- MongoDB
